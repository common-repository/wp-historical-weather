Historical Weather plugin for wordpress using Wunderground API.

== Description ==
Plugin for wordpress to show the weather from any date anywhere on your site using a shotcode [hw].  The site connects to Wunderground\'s JSON API.  The shortcode displays three times of the day: 6AM, Noon, 6PM.  The plugin displays relevant icons using a font instead of an image.  Font icon credit goes to Erik Flowers:
https://erikflowers.github.io/weather-icons/

Plugin works with any theme, no additional styling required.

== Installation ==
1.  Download Zip
2.  Upload plugin folder to uploads directory or with plugin uploader
3.  Activate plugin
